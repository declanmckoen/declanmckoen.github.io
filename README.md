# Declan's Portfolio
This project is a personal portfolio website developed using HTML, CSS, and JavaScript. The portfolio showcases my projects, skills, and experience, providing a professional online presence. The website is hosted using GitHub Pages and can be viewed at declanmckoen.github.io

## Repository Organization
- index.html: The main HTML file that structures the content of the portfolio.
- styles/: Directory containing the CSS files for styling the portfolio.
- scripts/: Directory containing the JavaScript files for interactive features.

## View Website
To run this project locally, you will need a web browser that can view the HTML file.

The portfolio is hosted on GitHub Pages. Any changes pushed to the main branch of the repository will be automatically deployed. To access the live website, visit declanmckoen.github.io
